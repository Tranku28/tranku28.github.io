FotoGraf

Description:

A fast-paced 3D game developed for the RE-use Game Jam. You play as a street photographer racing against time to pay overdue rent. Snap as many photos as possible before the clock runs out—while dodging traffic and staying alive on the bustling streets.

Team size:

5

I was responsible for:
  • Player Movement
  • Vehicle Physics
  • Photo Shot Mechanics
  • Collision Handling
  • Main Menu UI Programming
  • Game Over UI Programming
